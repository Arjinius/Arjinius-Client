# Arjinius Client

A custom Mudlet client created for **New Duris**, focused on clarity, usability, and a cohesive dark fantasy aesthetic.

Arjinius Client replaces the default Mudlet layout with a structured, information-rich interface designed to improve moment-to-moment gameplay without automating or altering core mechanics.

---

## Features

- Custom UI framework with percentage-based scaling
- Three-column layout with GMCP integration
- Integrated zone and wilderness mapper
- Chat tabs (All / Tells / Nchat / Room / Guild / Group)
- Clickable room exits, mobs, items, and players
- Context menus for common actions
- Time-of-day display and status indicators

---

## Requirements

- **Mudlet 4.19.1 or newer**
- GMCP enabled (default on New Duris)
